# API Analysis & Testing Project

## Overview
This project demonstrates hands-on analysis and testing of REST APIs to understand request-response behavior, identify potential vulnerabilities, and evaluate data exposure risks.

## Key Activities
- Intercepted and analyzed HTTP/HTTPS requests and responses  
- Tested REST API endpoints using Burp Suite  
- Identified potential data exposure issues in API responses  
- Performed request and response manipulation to analyze behavior  

## Tools & Technologies
- Burp Suite  
- REST APIs  
- HTTP/HTTPS Protocols  

## Outcome
Gained practical understanding of API workflows, debugging techniques, and data validation concepts. Improved ability to analyze backend systems and identify issues in data handling.
