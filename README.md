# API Analysis & Testing Project

## Overview
This project focuses on analyzing REST APIs to understand request-response behavior, identify vulnerabilities, and evaluate data exposure risks in web applications.

## Objectives
- Understand API workflow and communication
- Analyze request and response structure
- Identify potential security issues in APIs

## Work Performed
- Analyzed 50+ API requests and responses using Burp Suite
- Intercepted HTTP/HTTPS traffic and examined data flow
- Identified excessive data exposure in API responses
- Performed request manipulation to test edge cases
- Documented findings with observations and impact

## Tools & Technologies
- Burp Suite
- REST APIs
- HTTP/HTTPS Protocols

## Key Learnings
- Understanding of API architecture and endpoints
- Improved debugging and problem-solving skills
- Knowledge of data validation and backend behavior
